I have solved this strange issue!
Consider this declaration of a source broker:
        {declarations, [
                {'exchange.declare', [
                    {exchange, <<"MyQ-exchange">>},
                    {type, <<"direct">>},
                    *auto_delete*
                    ]},
                {'queue.declare', [
                    {queue, <<"MyQ">>},
                    *{arguments,[{<<"x-expires">>, long, 15000}]}*
                ]},
                {'queue.bind', [
                    {exchange, <<"MyQ-exchange">>},
                    {queue,    <<"MyQ">>}
                ]}

        ]}]}, 

Removing those two lines it started to work! So strange...
