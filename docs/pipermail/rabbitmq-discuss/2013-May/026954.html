I got the problem

Instead of using
BROKER_URL = "amqp://celeryuser@localhost:5672/celeryvhost"

I must use

BROKER_HOST = "localhost"
BROKER_PORT = 5672
BROKER_USER = "celeryuser"
BROKER_PASSWORD = "celery"
BROKER_VHOST = "celeryvhost"
