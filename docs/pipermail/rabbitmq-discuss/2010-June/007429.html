Hi Matthew,

thanks for the iptables pointer! Had to use the real ip address rather
than 127.0.0.1 to make it work:

sudo iptables -t nat -A PREROUTING -p tcp -d 172.16.200.10 --dport 80 -
j DNAT --to 172.16.200.10:55672

for some reason!

Cheers,
Donovan.
