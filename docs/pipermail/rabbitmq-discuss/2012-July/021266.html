Hi, Vijay:

You may want to upgrade from 2.7.1.  The 2.7.1 Java client had a known and
now fixed memory leak that might account for what you're seeing:

https://bugzilla.rabbitmq.com/show_bug.cgi?id=24723

Best regards,
Jerry
