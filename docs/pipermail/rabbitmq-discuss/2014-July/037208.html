My rabbitmq config file looks like :
[{rabbit,        [{tcp_listeners,    [5672]},{ssl_listeners, [5671]}, 
                        {ssl_options, 
[{cacertfile,"/etc/rabbitmq/testca/cacert.pem"},

 {rabbitmq_mqtt, [{default_user,     <<"guest">>},
