We are pleased to announce that the Spring AMQP (for Java) 1.0 project has reached RC1 status!

The Release Notes are here: https://jira.springsource.org/secure/ReleaseNote.jspa?projectId=10450&version=11821

The distribution zip is here: http://www.springsource.com/download/community?project=Spring%20AMQP

The JARs are available in the SpringSource Milestone Maven repository: http://shrub.appspot.com/maven.springframework.org/milestone/org/springframework/amqp/

As always, all other resource links (reference manual, javadoc, forum, etc.) are available on the homepage: http://www.springsource.org/spring-amqp

Please try it out! We'd like to get as much feedback as possible as we drive toward GA.

Cheers,
