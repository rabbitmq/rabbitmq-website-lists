ol of consumers (perhaps one for each stock, but less will work too), then set the stock name as the value of the _HQ_GROUP_ID property.<o:p></o:p></p><p class=MsoNormal><o:p>&nbsp;</o:p></p><p class=MsoNormal>With best regards,<o:p></o:p></p><p class=MsoNormal>Anand<o:p></o:p></p></div></body></html>
