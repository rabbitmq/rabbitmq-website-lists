<div>After further exploring it appears that you do this:</div>
<div>Using the BlockingConnection:</div>
<div>Receive Message</div>
<div>Run Handler to process message which can immediately ACK the message.</div>
<div>When Handler returns, next message is received from RabbitMQ.</div>
<div><br>
</div>
<div>Simpler!</div>
