<br/>> Also setting this relative to memory sets really "low watermark". For example, If you have 16G RAM and 100G Hard disk. Setting this relative to RAM makes low watermark set at 16GB. This means that Publishers are blocked when your available disk space approaches 84GB (100-16). So I set this value to 0.1 and now publishers are not blocking in case of backup until 98.4 (100-1.6). HTH
