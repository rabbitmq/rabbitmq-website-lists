<font face="trebuchet ms,sans-serif">Hi, I&#39;m glad to announce the availability of ManagedShovel, an external mechanism to replace the built-in shovel addin.</font><div><font face="&#39;trebuchet ms&#39;, sans-serif"><br>

</font><div><font face="trebuchet ms,sans-serif">Its purpose is to provide the same features of the official addin and add some more. Currently the only additional feature is a max hops option, which helps prevent messages from going in loops indefinitely.</font></div>

<div><span style="font-family:&#39;trebuchet ms&#39;,sans-serif">The project uses the .NET client library to connect to RabbitMQ brokers and can be configured using a fluent interface, which makes it easy to host it anywhere from a console application to a windows service.</span></div>

<div><span style="font-family:&#39;trebuchet ms&#39;,sans-serif">With the project comes a demo application which shows how to configure it, although all the configuration options are semantically equivalent to those of the original addin.</span></div>

<div><span style="font-family:&#39;trebuchet ms&#39;,sans-serif"><br></span></div><div><span style="font-family:&#39;trebuchet ms&#39;,sans-serif">The source code is hosted on github at <a href="https://github.com/simoneb/ManagedShovel">this page</a>. Please be aware that this is pre-release software, and has not been tested in production.</span></div>
