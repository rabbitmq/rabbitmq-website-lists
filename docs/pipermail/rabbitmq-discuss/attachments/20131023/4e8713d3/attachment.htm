<div dir="ltr"><div>I&#39;m happy to announce the initial release of Lyra: a high availability client for RabbitMQ on the JVM!</div><div><br></div><div>--</div><div><br></div><div>Lyra is a RabbitMQ client that helps you achieve high availability in your services by embracing failure, allowing for AMQP resources such as connections, channels and consumers to be automatically recovered when server or network failures occur. It also supports automatic invocation retries, and exposes a simple, lightweight API built around the Java AMQP-client library.</div>
