The error is that the frame is too large. See the &quot;frame_max&quot;<br>
configuration item in <a href="http://www.rabbitmq.com/configure.html" target="_blank">http://www.rabbitmq.com/configure.html</a>. Either set<br>
it too a larger value or to 0 for unlimited.<br>
<br>
The maximum frame size was not strictly enforced before v3.0.0, which is<br>
why the issue appeared after upgrading. See item 24991 in<br>
<a href="http://www.rabbitmq.com/release-notes/README-3.0.0.txt" target="_blank">http://www.rabbitmq.com/release-notes/README-3.0.0.txt</a><br>
