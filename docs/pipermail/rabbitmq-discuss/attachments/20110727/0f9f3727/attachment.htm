<P>I'm sorry our troubleshooting instructions weren't able to help you.</P>
<P>&nbsp;</P>
<P>Does the rabbitmq service still appear in your list of services?</P>
<P>Is epmd.exe still running in your processes? You may like to kill that.</P>
<P>Did you make any change to the cookie or was that ok?</P>
<P>&nbsp;</P>
<P>It sounds like you had an older version already installed. Is that correct?</P>
<P>If so, you may need to install the older version combination in order to stop and remove the old service.</P>
<P>After removing the service you should be able to uninstall rabbitmq, remove directories (including mnesia and logs if you want) and uninstall erlang, and remove any&nbsp;environment variables&nbsp;you dont want (eg from path and ERLANG_HOME).</P>
<P>Double check there is no RabbitMQ service present and that epmd.exe is not running. You may wish to reboot.</P>
<P>&nbsp;</P>
<P>After that a clean&nbsp;fresh install should work ok. Our "experimental" installer works quite well, just ensure you stop, remove and start the service again (or reboot) after adding plugins. Also remember you may need to run some steps as administrator (elevated permission).</P>
