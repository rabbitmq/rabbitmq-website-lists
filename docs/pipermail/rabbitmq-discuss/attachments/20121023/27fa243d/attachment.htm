    Post me the *exact* commands you're using on each node, in sequence
    please. If you follow the 'Breaking up a cluster' transcript from

    *any* problems. In particular, note the use of 'force_reset' on the
    surviving node, so that it does not attempt to connect to nodes that

    still present on the extant node we're now resetting.<br>
