<DIV>&nbsp;</DIV>
<DIV>RabbitMQ sender&nbsp; -&gt; Source RabbitMQ server&nbsp; -&gt; RabbitMQ 
shovel -&gt; Destination RabbitMQ server -&gt; RabbitMQ receiver</DIV>
<DIV>192.168.1.10&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;192.168.1.10&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;

192.168.1.12</DIV>
<DIV>&nbsp;</DIV>
<DIV>1: First we didn't start RabbitMQ receiver. The messages are stored in the 
"Destination RabbitMQ server".</DIV>
<DIV>2: Then we stop the RabbitMQ sender. And we start RabbitMQ receiver. At 
this time the consuming rate is around 500 messages per second.</DIV>
<DIV>3: Now we restart the RabbitMQ sender again. However, the RabbitMQ receiver

<DIV>&nbsp;</DIV>
<DIV>So from our view, whether&nbsp;the messages keep coming into the
