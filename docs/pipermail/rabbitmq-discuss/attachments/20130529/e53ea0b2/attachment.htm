You need to set a &#39;receipt&#39; header on the SEND frame, as per the spec - <a href="http://stomp.github.io/stomp-specification-1.1.html#Header_receipt" target="_blank">http://stomp.github.io/stomp-<u></u>specification-1.1.html#Header_<u></u>receipt</a>.<br>

<br>
Btw, a) there is no point setting *both* persistent:true and delivery-mode:2, and b) receipts are orthogonal to persistence, i.e. whether receipts are issued depends solely on whether the SEND frame contains a &#39;receipt&#39; header.<br>
