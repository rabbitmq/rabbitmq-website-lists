Plugins show version 0.0.0 if they&#39;re self-compiled and you didn&#39;t specify a VERSION when bulding them.<br>
<br>

<br>
I think you&#39;re running self-compiled plugins from hg default against the 2.3.1 server release - certainly there were some API changes in this area. This won&#39;t work - plugins tend to be bound tightly to a given server release.<br>

<br>
So either download binary plugins from <a href="http://www.rabbitmq.com/releases/plugins/v2.3.1/" target="_blank">http://www.rabbitmq.com/releases/plugins/v2.3.1/</a> or update all your plugin repos to tag rabbitmq_v2_3_1, make clean everywhere, and rebuild.<br>
