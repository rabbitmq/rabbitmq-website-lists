The admin flag has been replaced by a &quot;tags&quot; field for users. Users can be given arbitrary tags within rabbitmq-server. rabbitmq-management then checks for the following tags:<br>
<br>
&quot;administrator&quot; (do everything, same as admin before)<br>
&quot;monitoring&quot; (look at everything, but only touch your own stuff)<br>
&quot;management&quot; (limited access to mgmt, same as non-admin before)<br>
<br>
Note also that by giving a user no tags you can lock them out of mgmt completely. This would be useful if (for example) you use secret queue names as capabilities.<div class="im"><br></div></blockquote><div><br></div><div>
