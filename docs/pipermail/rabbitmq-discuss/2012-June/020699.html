As you said, this log does not contain anything useful, which is
unusual. Check whether there was enough diskspace to write to the
logfile. Alternative check for the presence of an erl_crash.dump in the
home folder of the rabbitmq user. You should also check whether the sasl
log contains entries over the relevant period.
