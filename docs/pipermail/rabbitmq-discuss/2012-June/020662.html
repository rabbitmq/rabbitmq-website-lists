On 14/06/12 21:00, Jeff wrote:
> that being said - is there any situation where rabbit would drop all
> connections at once?

Check for entries in the logfile that say "connection_closed_abruptly".
If you see many of these simultaeously then you can be fairly certain
that a network interruption is the cause rather than the broker.
