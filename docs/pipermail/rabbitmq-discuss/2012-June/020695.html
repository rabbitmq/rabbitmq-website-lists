The most common reason for this in version 2.8.2 is not having enough
disk space. Check if the broker logfile contains a message to that
effect. You can solve the problem by requiring less free disk space or
by making sufficient disk space available on the partition that the
broker uses. See

http://www.rabbitmq.com/memory.html#diskfreesup
