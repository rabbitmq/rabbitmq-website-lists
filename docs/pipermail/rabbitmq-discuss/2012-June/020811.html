Yesterday in the Bunny 0.8.0 release announcement I've stated that 0.8.0 is the last release to support AMQP 0.8.
This is not correct: 0.7.9 is the last release. 0.8.0 already only supports AMQP 0.9.1.

This means Bunny 0.8.0 requires RabbitMQ 2.0 and later. We will update the README to say so
very soon.
