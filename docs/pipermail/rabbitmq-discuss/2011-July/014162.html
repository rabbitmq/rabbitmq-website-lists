plugin code dealing with heart-beats. The code contains:

  ensure_heartbeats(...) -> ...
    catch rabbit_net:send(Sock, <<0>>)

The STOMP 1.1 spec states that newlines (not nulls) can be used for
heart-beats. Shouldn't it be <<10>> instead of <<0>>?
