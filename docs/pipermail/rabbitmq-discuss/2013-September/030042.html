March Hare (née HotBunnies) 2.0.0.rc1 is released [1].
With this version the project changes name from Hot Bunnies
(which has a sexist meaning, chosen unintentionally) to March Hare.

RC1 is ready for developers to start migrating to it while we review and extend
doc guides. Note that there are some breaking public API changes.

Release notes:
http://blog.rubyrabbitmq.info/blog/2013/09/11/march-hare-2-dot-0-0-dot-rc1-is-released/

1. https://rubygems.org/gems/march_hare/versions/2.0.0.rc1-java
