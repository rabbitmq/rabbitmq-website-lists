Hello RabbitMQ community!

I just wanted to let you know that we're running an infrastructure
competition in Ubuntu:

https://github.com/juju/charm-championship

We've got over $60k in prizes that we're giving out, and I wanted to
point out that we have a nice charm for Rabbit. A charm is a script
that deploys and manages a service:

https://jujucharms.com/charms/precise/rabbitmq-server

We're using this charm in production with our OpenStack deployment; so
we're confident that it's robust enough for people using Rabbit on
Ubuntu to give it a shot. We have over 130 services integrate with in
the Charm Store so I was thinking that if you're using Rabbit on
Ubuntu already and you're deploying it with other services then you
might want to investigate the contest and see if you want to earn some
money too. :)

The contest runs through 22 October, thanks!
