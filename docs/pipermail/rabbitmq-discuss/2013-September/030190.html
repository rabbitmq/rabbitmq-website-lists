Does it change for the better or the worse? What change did you make?

When we developed the flow control feature we considered making this 
tunable, but there seemed to be a fairly broad plateau for the initial 
credit values with broadly similar performance, so we just picked values 
in the middle of that range and stuck with it. If you have a scenario 
where tuning this value does something useful I'd like to hear about it...
