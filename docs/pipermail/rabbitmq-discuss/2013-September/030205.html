RabbitMQ supports multi-tenancy via vhosts (similar to Apache or Nginx vhosts).

Graylog2 probably uses its own vhost and by default the UI presents you the default
vhost (/). See the switcher in the top right corner:

https://www.monosnap.com/image/7aINJygDxDsomYVqwb9AvTw7s

and select the vhost you need. Note that you may need to log out and
log back in with a user that has access to the Graylog2 vhost.
