On 13/09/13 05:10, Robert Gilland wrote:
> Every time the system restarts I lose all my enabled plugins what could
> possibly be happening?

The enabled plugins file is stored in %RABBITMQ_BASE% - by default, 
%APPDATA%\RabbitMQ\enabled_plugins.

So is %APPDATA% somehow non-persistent across reboots on your machine? 
If so you probably want to point %RABBITMQ_BASE% elsewhere since that 
contains the message store too!

See http://www.rabbitmq.com/relocate.html for more information.
