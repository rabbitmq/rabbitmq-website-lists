I'm happy to report to all Logstash users on this list that
Logstash 1.2 is out and it features a completely rewritten RabbitMQ plugin:

https://github.com/logstash/logstash/blob/master/CHANGELOG

It should be backwards compatible configuration-wise. Note that it now
requires you to run RabbitMQ 2.0 or later.

Among many improvements in this version, you will see x4-6 better
throughput for RabbitMQ output and x3-4 for RabbitMQ input (with a single instance)
on JRuby. There will be some improvements on CRuby, too, just not as significant.
