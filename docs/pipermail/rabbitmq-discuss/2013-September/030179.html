I wrote an application based on EasyNetQ which is available for use by anyone.

https://github.com/chrisedebo/RabbitMQ.LoadTest

It will require a set of text files which are randomly passed across the message bus by the publishers. The subscriber will retrieve these messages and output a count to the console.

Have a look and let me know if you need any assistance with it.
