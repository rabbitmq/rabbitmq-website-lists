Would like to start the discussion on what ( if any ) the Mobile Client for 
RabbitMQ should look like in terms of functionality / features.

For me I would like to see the following:

- Overview ( Something like the management plugin overview tab )
  * minus the import / export functionality

- Connections

- Queues
  * With get and submit message functionality

- Admin
  * User creation/permissions

Would also like to see some sort of alert functionality ( Push 
Notifications ) for:

- Node Down
- Server Restart
- Disc Space
- RAM usage

if anyone else has some ideas as to what should be in the mobile 
app/client, feel free to voice them here
