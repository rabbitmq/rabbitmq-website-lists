Well, every release has bug fixes, so in general I would suggest you 
upgrade to the latest version (3.1.5 at time of writing).

But there are a few incompatibilities between 2.x and 3.x (see 
https://www.rabbitmq.com/blog/2012/11/19/breaking-things-with-rabbitmq-3-0/) 
so if upgrading to 3.1.x will take too long then 2.8.7 should be the 
same as 2.8.2, just with fewer bugs.

Cheers, Simon
