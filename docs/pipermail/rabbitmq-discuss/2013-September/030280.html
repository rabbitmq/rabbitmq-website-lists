amqp gem 1.1.0.rc1 is released.

This version removes amq-client dependency, introduces some usability improvements
and removes more long time deprecated API elements.

Please give this version a try. A few things will be added before 1.1.0 final,
but we need some feedback about stability after substantial internal changes.

Release notes:
http://blog.rubyrabbitmq.info/blog/2013/09/19/amqp-gem-1-dot-1-0-rc1-is-released/
