Release: RabbitMQ .Net/C# AMQP client library 1.4.0
Status : beta

Release Highlights
================== 

bug fixes
- rationalised shutdown protocol which included fixing various
  race conditions
- removed dependency on read timeout of .Net sockets to avoid
  problem with unusable state (socket closed)

enhancements
- changed the default protocol to the one RabbitMQ broker
  currently uses
- better exception hierarchy
- better documentation of included examples, client configuration
