The RabbitMQ team is pleased to announce the release of RabbitMQ 2.8.3.

This is primarily a bugfix release, with some minor enhancements (make 
disk space monitoring less intrusive by default, improve SSL performance 
with the Java client).

See the attached release notes for more details.

The new release can be downloaded from:

http://www.rabbitmq.com/download.html

As always, we welcome any questions, bug reports, and other feedback on
this release, as well as general suggestions for features and
enhancements in future releases. Mail us via the RabbitMQ discussion
list.

Regards,
                      The RabbitMQ Team
                      (http://www.rabbitmq.com)
